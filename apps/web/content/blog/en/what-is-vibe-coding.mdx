## What Is Vibe Coding?

**Vibe coding** is a new approach to programming where you give instructions to AI in plain language to write code. You don't need to know coding syntax — just say "make a login page" and the AI generates the code for you.

The biggest difference from traditional coding is that **there's no language barrier**. You can explain what you want in English, Korean, Japanese, or any language.

## Why Vibe Coding Now?

Since 2025, AI tools like Claude, ChatGPT, and Cursor have rapidly improved, opening up app development to non-developers.

- **Got an idea?** You can build a prototype yourself
- **No hiring needed** for simple tools and automations
- **Low learning curve** means anyone can start

## Tools You Need for Vibe Coding

To get started, you need a basic development environment:

1. **Git** — Version control for your code
2. **Node.js** — JavaScript runtime
3. **VS Code** — Code editor
4. **AI tools** — Claude, Cursor, GitHub Copilot, etc.
