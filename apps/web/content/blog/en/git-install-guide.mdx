## What Is Git?

**Git** is a tool that tracks changes in your code. Like pressing a "save" button, it records each version of your code so you can go back to any previous state.

Whether you're vibe coding or doing traditional development, Git is an essential tool used in almost every project.

## Installing Git on Windows

Open PowerShell and run:

```powershell
winget install --id Git.Git -e --source winget
```

After installation, close and reopen your terminal, then verify with `git --version`.

## Installing Git on macOS

Open Terminal and run:

```bash
xcode-select --install
```

This installs Xcode Command Line Tools, which includes Git.

## Initial Git Setup

After installation, set your name and email:

```bash
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
```
