## VS Code란?

**Visual Studio Code(VS Code)**는 마이크로소프트가 만든 무료 코드 편집기입니다. 가볍고 빠르면서도 강력한 기능을 제공해 전 세계에서 가장 많이 사용되는 편집기입니다.

바이브코딩에서도 VS Code는 핵심 도구입니다. AI 코딩 도구들이 VS Code 확장 프로그램으로 제공되기 때문입니다.

## Windows에서 VS Code 설치하기

PowerShell을 열고 다음 명령어를 실행합니다:

```powershell
winget install --id Microsoft.VisualStudioCode -e --source winget
```

## macOS에서 VS Code 설치하기

터미널을 열고 Homebrew로 설치합니다:

```bash
brew install --cask visual-studio-code
```

Homebrew가 없다면 먼저 설치해야 합니다:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## 바이브코딩을 위한 추천 확장 프로그램

VS Code를 설치한 후 다음 확장 프로그램을 추가하면 바이브코딩이 훨씬 편해집니다:

1. **Claude Dev / Cline** — Claude AI와 대화하며 코딩
2. **GitHub Copilot** — AI 코드 자동 완성
3. **Korean Language Pack** — VS Code 한국어 지원
4. **ESLint** — 코드 품질 검사
5. **Prettier** — 코드 자동 정리

## 터미널에서 VS Code 열기

설치 후 터미널에서 `code .`을 입력하면 현재 폴더를 VS Code로 열 수 있습니다. 이 기능은 프로젝트 작업 시 매우 편리합니다.
