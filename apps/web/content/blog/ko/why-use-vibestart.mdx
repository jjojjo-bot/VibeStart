## 개발 환경 설정, 뭐가 그렇게 어려운 걸까?

"AI로 코딩하고 싶다"는 마음으로 시작했는데, 정작 코딩을 시작하기도 전에 막히는 경우가 많습니다.

- Git을 설치하라는데 **터미널이 뭔지 모르겠다**
- Node.js 버전이 여러 개인데 **뭘 깔아야 하지?**
- VS Code를 깔았는데 **확장 프로그램은 뭘 설치해야 하지?**
- 블로그마다 설명이 다르고, **내 OS에 맞는 건지 모르겠다**

구글에서 "Git 설치 방법"을 검색하면 수십 개의 글이 나오지만, 절반은 오래된 정보이고, 나머지 절반은 이미 개발 경험이 있는 사람 기준으로 쓰여 있습니다.

## VibeStart가 다른 점

### 1. OS를 선택하면 맞춤 명령어를 제공합니다

Windows인지 macOS인지 선택하면, 해당 OS에서 바로 실행할 수 있는 명령어만 보여줍니다. "이 명령어가 내 컴퓨터에서 되는 건가?" 고민할 필요가 없습니다.

### 2. 복사 버튼 하나로 끝납니다

명령어를 직접 타이핑할 필요가 없습니다. **복사 버튼을 누르고 터미널에 붙여넣기만 하면 됩니다.** 오타로 인한 에러가 발생하지 않습니다.

### 3. 순서가 정해져 있습니다

"Git을 먼저 깔아야 하나? Node.js를 먼저 깔아야 하나?" VibeStart는 설치 순서를 자동으로 정해줍니다. 이전 단계가 완료되어야 다음 단계가 활성화되기 때문에, 순서를 잘못 따라가는 일이 없습니다.

### 4. 불필요한 설명이 없습니다

"Git의 내부 작동 원리는..." 같은 설명은 나중에 알아도 됩니다. VibeStart는 **지금 당장 실행해야 하는 것**만 보여줍니다. 이해하기 전에 일단 돌아가는 환경을 만드는 게 먼저입니다.

### 5. 완료하면 바로 코딩을 시작할 수 있습니다

모든 단계를 마치면 Git, Node.js, VS Code가 설치되고 첫 번째 프로젝트 폴더까지 생성됩니다. AI 도구를 열고 "로그인 페이지 만들어줘"라고 말하면 바로 바이브코딩을 시작할 수 있습니다.

## 실제로 얼마나 걸리나요?

대부분의 사용자가 **10분 이내에** 모든 설치를 완료합니다. 직접 검색하면서 설치하면 최소 30분~1시간, 에러가 나면 반나절도 걸릴 수 있는 작업입니다.

## 어떤 도구가 설치되나요?

VibeStart를 통해 설치되는 도구는 다음과 같습니다:

| 도구 | 용도 |
|------|------|
| **Git** | 코드 버전 관리 |
| **Node.js** | JavaScript 실행 환경 |
| **VS Code** | 코드 편집기 |
| **Next.js 프로젝트** | 첫 번째 웹 프로젝트 생성 |

모두 전 세계 개발자가 가장 많이 사용하는 표준 도구입니다.
