## 바이브코딩이란?

**바이브코딩(Vibe Coding)**은 AI에게 자연어로 지시해서 코드를 작성하는 새로운 프로그래밍 방식입니다. 코딩 문법을 몰라도 "로그인 페이지를 만들어줘"처럼 말하면 AI가 코드를 생성해줍니다.

기존 코딩과 가장 큰 차이는 **언어의 장벽이 없다**는 점입니다. 한국어, 영어, 일본어 등 어떤 언어로든 AI에게 설명하면 됩니다.

## 왜 지금 바이브코딩인가?

2025년 이후 Claude, ChatGPT, Cursor 같은 AI 도구가 급격히 발전하면서 비전공자도 앱을 만들 수 있는 시대가 열렸습니다.

- **아이디어만 있으면** 프로토타입을 직접 만들 수 있음
- **개발자 채용 없이** 간단한 도구를 만들 수 있음
- **학습 곡선이 낮아** 누구나 시작할 수 있음

## 바이브코딩에 필요한 도구

시작하려면 기본적인 개발 환경이 필요합니다:

1. **Git** — 코드 버전 관리 도구
2. **Node.js** — JavaScript 실행 환경
3. **VS Code** — 코드 편집기
4. **AI 도구** — Claude, Cursor, GitHub Copilot 등

이 도구들을 설치하는 게 바로 첫 번째 장벽입니다. VibeStart는 이 과정을 복사 붙여넣기만으로 해결해드립니다.
