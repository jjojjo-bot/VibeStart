## 📌 이 글을 읽어야 하는 이유

바이브코딩 도구는 종류마다 설치 조건이 다릅니다. 어떤 도구는 앱만 설치하면 바로 되고, 어떤 도구는 Node.js와 API 키가 필요합니다. 도구를 설치하려다 "이게 왜 안 되지?"에서 막히는 경우가 대부분 사전 요구사항을 빠뜨렸기 때문입니다.

이 글은 주요 바이브코딩 도구 5개의 설치 요구사항을 한 곳에 정리합니다. 자신이 쓸 도구가 정해졌다면 필요한 부분만 확인하고, 아직 정하지 않았다면 전체를 훑어보면서 어떤 도구가 자신의 환경에 맞는지 판단할 수 있습니다.

## 🖱️ Cursor

### 설치 요구사항

| 항목 | 내용 |
|------|------|
| OS | Windows 10 이상 또는 macOS 12 (Monterey) 이상 |
| RAM | 8GB 이상 권장 |
| 디스크 | 약 500MB |
| 계정 | Cursor 계정 (이메일/Google/GitHub 가입) |

### 사전 설치가 필요한 것

Cursor 자체는 별도 사전 설치 없이 동작합니다. 하지만 바이브코딩으로 웹 프로젝트를 만들려면 아래 도구가 필요합니다.

- **Node.js** (v18 이상): 프로젝트 생성 및 실행에 필요
- **Git**: 버전 관리 및 프로젝트 초기화에 필요

### 비용

- Hobby (무료): 월 2,000 자동완성, 50 느린 프리미엄 요청
- Pro ($20/월): 무제한 자동완성, 500 빠른 프리미엄 요청

### 설치 가이드

→ [Cursor 설치 방법과 첫 프로젝트 만들기](/blog/cursor-install-first-project)

## 🤖 Claude Code

### 설치 요구사항

| 항목 | 내용 |
|------|------|
| OS | Windows 10 이상, macOS 12 이상, Linux |
| Node.js | v18 이상 (필수 — 설치 자체에 필요) |
| RAM | 4GB 이상 |
| 계정 | Anthropic API 계정 또는 Claude 구독 |

### 사전 설치가 필요한 것

Claude Code는 Node.js 기반으로 동작하기 때문에 **Node.js가 반드시 먼저 설치돼 있어야** 합니다.

- **Node.js** (v18 이상): `npm install -g` 명령어 실행에 필수
- **Git** (권장): 프로젝트 관리에 유용

### API 키 또는 구독

| 방법 | 비용 | 설정 난이도 |
|------|------|-----------|
| Anthropic API 키 | 종량제 ($1~$5/월 입문) | 중간 (키 발급 + 환경 변수 설정) |
| Claude Max 구독 | $100~$200/월 | 쉬움 (로그인만) |

입문자에게는 API 키 방식을 권장합니다. 비용이 적고 사용량을 직접 관리할 수 있습니다.

### 설치 가이드

→ [Claude Code 설치부터 첫 코딩까지 비전공자 가이드](/blog/claude-code-beginner-guide)

## 🧑‍✈️ GitHub Copilot

### 설치 요구사항

| 항목 | 내용 |
|------|------|
| 에디터 | VS Code, JetBrains IDE, Neovim 등 |
| 계정 | GitHub 계정 (필수) |
| Git | 필수 (GitHub 연동) |
| RAM | 8GB 이상 권장 |

### 사전 설치가 필요한 것

- **VS Code** 또는 지원하는 에디터: Copilot은 에디터 확장 프로그램으로 동작합니다.
- **Git**: GitHub 연동에 필수
- **GitHub 계정**: Copilot을 활성화하려면 GitHub 계정이 필요합니다.

### 비용

- Free: 월 2,000 자동완성, 50 채팅 메시지
- Pro ($10/월): 무제한 자동완성, 무제한 채팅
- Pro+ ($39/월): 에이전트 모드 포함

### 특징

Copilot은 VS Code 안에서 확장 프로그램으로 동작하므로, 기존 VS Code 사용자가 가장 쉽게 추가할 수 있는 AI 도구입니다. 다만 단독으로 프로젝트를 생성하는 기능은 Cursor나 Claude Code보다 제한적입니다.

## 🧪 Google AI Studio

### 설치 요구사항

| 항목 | 내용 |
|------|------|
| 필요 소프트웨어 | 웹 브라우저 (Chrome 권장) |
| 계정 | Google 계정 |
| 설치 | 불필요 (웹 기반) |

### 사전 설치가 필요한 것

없습니다. 웹 브라우저만 있으면 됩니다. 별도 소프트웨어 설치가 필요 없기 때문에 진입 장벽이 가장 낮습니다.

### 비용

- 무료: 일일 요청 한도 내 사용 가능
- 유료 (Gemini API): 사용량에 따라 과금

### 특징

Google AI Studio는 브라우저에서 바로 AI와 대화하며 코드를 생성할 수 있습니다. 별도 설치가 필요 없어 가장 빠르게 시작할 수 있지만, 생성된 코드를 로컬 프로젝트에 적용하려면 복사-붙여넣기가 필요합니다. 에디터와 직접 연동되지 않는 점이 다른 도구와의 차이입니다.

## 🏄 Windsurf

### 설치 요구사항

| 항목 | 내용 |
|------|------|
| OS | Windows 10 이상, macOS 12 이상 |
| RAM | 8GB 이상 권장 |
| 디스크 | 약 500MB |
| 계정 | Windsurf 계정 |

### 사전 설치가 필요한 것

Cursor와 마찬가지로 앱 자체는 독립적으로 설치됩니다. 프로젝트 작업에는 Node.js와 Git이 필요합니다.

- **Node.js** (v18 이상): 프로젝트용
- **Git**: 프로젝트용

### 비용

- Free: 월 크레딧 제공 (자동완성 + AI 채팅)
- Pro ($15/월): 더 많은 크레딧과 모델 선택

### 특징

Windsurf는 Cursor와 유사한 AI 코드 에디터입니다. VS Code 기반이며, AI가 코드 맥락을 파악해서 제안하는 "Flow" 기능이 특징입니다. Cursor와 비슷한 사용 경험을 원하지만 다른 옵션도 살펴보고 싶다면 고려해볼 만합니다.

## 🗺️ 공통으로 필요한 것

어떤 도구를 선택하든 바이브코딩으로 웹 프로젝트를 만들려면 아래 도구가 필요합니다.

### 필수

| 도구 | 용도 | 확인 명령어 |
|------|------|-----------|
| Node.js (v18+) | 프로젝트 생성 및 실행 | `node --version` |
| Git | 버전 관리, 프로젝트 초기화 | `git --version` |

### 권장

| 도구 | 용도 |
|------|------|
| VS Code 또는 Cursor | 코드 편집 및 확인 |
| 터미널 사용법 | 명령어 실행, 프로젝트 관리 |
| GitHub 계정 | 코드 저장 및 공유 |

Node.js와 Git이 설치돼 있다면 어떤 도구든 바로 시작할 수 있습니다. 이 두 가지가 바이브코딩의 기본 인프라입니다.

## 🏁 도구 선택 플로차트

자신의 상황에 맞는 도구를 빠르게 찾아보세요.

**터미널을 써본 적 없다면** → Cursor 또는 Windsurf로 시작
**터미널은 괜찮고, AI에게 많이 맡기고 싶다면** → Claude Code
**VS Code를 이미 쓰고 있고, 가볍게 추가하고 싶다면** → GitHub Copilot
**아무것도 설치하지 않고 바로 시작하고 싶다면** → Google AI Studio
**잘 모르겠다면** → Cursor 무료 플랜으로 시작 (가장 균형 잡힌 선택)

어느 도구로 시작해도 나중에 다른 도구로 전환하거나 병행할 수 있습니다. 첫 선택에 너무 고민하지 마세요.
